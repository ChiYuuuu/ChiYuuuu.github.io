Hexo + github 搭建博客

hexo中文官网 :https://hexo.io/zh-cn/

一、准备

  1. github 账号
  2. node 环境

二、搭建

  1. 登录github账号访问 https://github.com/new创建一个新的仓库

    Repository name : 你的用户名.github.io

  2. 安装 hexo npm install -g hexo

  3. 先后执行下面两条命令.

       
hexo init
hexo s

  4. 访问localhost:4000即可看到一个初始的hexo博客网站

  5. github 搜索 hexo-theme 选择一个好看的主题，我选择的是 https://github.com/blinkfox/hexo-theme-matery

  6. 主题里都有安装文档,按照文档操作即可,执行 hexo g生成 静态页面

  7. 将 public目录下的文件上传至 创建好仓库的根目录访问 你的用户名.github.io 就大功告成了
