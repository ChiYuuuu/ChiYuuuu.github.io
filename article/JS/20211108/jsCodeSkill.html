常用的JS技巧

判断

1. if else 判断

简写 if else

// 长
let test: boolean;
if (x > 100) {
  test = true;
} else {
  test = false;
}
// 短
let test = (x > 10) ? true : false;
// 也可以直接这样
let test = x > 10;

多表达式多 if 判断

在数组中存储多个值，并且可以使用数组include方法

// 长
if (x === 'abc' || x === 'def' || x === 'ghi' || x ==='jkl') {
  //logic
}
// 短
if (['abc', 'def', 'ghi', 'jkl'].includes(x)) {
  //logic
}

运算符

1. &&

如果仅在变量值为 true 的情况下才调用函数，则可以使用 && 运算符。

// 长 
if (test1) {
 callMethod(); 
} 
// 短 
test1 && callMethod();
